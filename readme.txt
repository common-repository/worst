Wordpress Shipping Costs

== Description ==
[**WORST**](https://github.com/Kristories/WORST) can help you deal with the shipping costs.

== Installation ==
[**Installation**](http://wordpress.org/plugins/worst/screenshots/).

== Screenshots ==
1. Install **WORST**.
2. Activate **WORST** plugin in the "Plugins" admin panel using the "Activate" link.
3. Go to **Settings** -> **WORST**.
4. Save settings.
5. Create a page with the name you want, but you need to change slug with "**worst**".
6. Go to **Shipping Costs**.
7. You can proceed with entering the desired content.
8. Go to *http://your.lab/worst*
9. Testing.. That's it. You're ready to go!
